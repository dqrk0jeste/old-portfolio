Cube w/ Friends

(due to free hosting on render.com it will take some time to initially load the site sometimes)
This is my first big project, made from scratch using Node.js (Express and Socket.io) and MongoDB on backend and Vanilla JavaScript on the front. As speedcubing is one of my favorite hobbies I made this web app in order to make it easy for me and others to solve together. It's a real time app with round-based rooms. Each round everyone is given the same cube scramble. After you time your solve you can submit your result, which is then shared with everyone in the same room, in real time, using Web Sockets. After the round is over, final results are displayed and the next round starts. I intend to continue working on this app in the future, to implement other features I have in mind, like admin control over the room users, different modes, comparing averages, leaderboards etc.
